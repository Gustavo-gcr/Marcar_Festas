README
Passos para configuração do projeto
1. Clone este repositório em sua máquina local
2. Certifique-se de ter o ambiente de desenvolvimento .NET configurado
3. Clone o banco de dados no seu servidor local
4. Configure a conexão com o banco de dados
4.1. No código da aplicação, você precisará ajustar a string de conexão na classe ConexaoBanco. Abra o arquivo ConexaoBanco.cs e localize a propriedade data_source. Altere a string de conexão conforme necessário, inserindo seu usuário e senha para acessar o servidor local.
5. Execute o projeto em seu ambiente de desenvolvimento.

Observação:
O Sistema de Gerenciamento de Banco de Dados (SGBD) utilizado no projeto, e no qual a lista de comandos se baseia, é o MySQL Workbench.
